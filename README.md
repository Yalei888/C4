# C51 模拟 C4 炸弹

基于8051单片机的CS:GO风格C4炸弹模拟器。

## 功能特性

- **密码输入**：通过4×4矩阵键盘输入7位密码
- **倒计时**：密码正确后启动40秒倒计时
- **C4拆除**：倒计时期间可按键拆除C4
- **音效提示**：蜂鸣器提供倒计时和操作音效
- **LCD显示**：LCD1602实时显示状态和倒计时

## 游戏规则

| 场景 | 结果 |
|------|------|
| 倒计时结束 | 💥 C4爆炸，T（恐怖分子）获胜 |
| 成功拆除C4 | 🛡️ CT（反恐精英）获胜 |
| 密码错误 | ❌ 显示"ERRO!"并重置 |

## 硬件连接

### LCD1602
- RS → P2.6
- RW → P2.5
- EN → P2.7
- D0-D7 → P0

### 矩阵键盘
- 行线 → P1.4-P1.7
- 列线 → P1.0-P1.3

### 蜂鸣器
- 信号端 → P1.5

### 按键（拆除按钮）
- K1 → P3.1

## 默认密码

```
7 3 5 5 6 0 8
```

## 操作说明

1. 开机显示 "C4 Planting..."
2. 通过矩阵键盘输入7位密码
3. 按 **确认键（#）** 验证密码
4. 密码正确后开始40秒倒计时
5. 按 **K1键** 开始拆除C4（需持续按住）
6. 拆除进度完成后显示 "CT Win!"
7. 按 **清除键（*）** 可重置输入
